# AAAI Jordan Student Chapter — Official Website

The official website for the AAAI Jordan Student Chapter, an affiliate of the [Association for the Advancement of Artificial Intelligence](https://aaai.org). Founded 2026 at German Jordanian University, Amman, Jordan.

**Live site:** *[insert deployed URL]*

## Overview

A single-page site covering the chapter's mission, focus areas, upcoming events (including ICFT 2026), chapter leadership, and membership information. Built as a static HTML page — no build tools or frameworks required.

## Tech Stack

- Pure HTML / CSS / JavaScript
- Google Fonts (Syne, Space Mono, Inter)
- Custom cursor, scroll-reveal animations, glassmorphism cards
- Fully responsive

## Structure

```
├── index.html      # Full site (single file)
├── assets/         # Images and media
└── README.md
```

## Running Locally

Open `index.html` in any browser. No server or dependencies needed.

## Deployment

Deployed via GitHub Pages. Any push to `main` triggers a rebuild automatically.
